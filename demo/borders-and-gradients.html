Borders & Gradients

CSS3 Trick and Treat

#the-button {
    width: 30px;
    height: 30px;
}

.border {
    border: solid 1px rgba(0,0,0,0.2);
}

.border-radius {
    -webkit-border-radius: 15px;
}

.background {
    background: -webkit-gradient(
        linear, left top, left bottom,
        from(#f0f0f0), to(#b8b8b8)
    );
}

.box-shadow {
    -webkit-box-shadow: #333 0 1px 1px;
}

.before:before {
    background: -webkit-gradient(
        linear, left top, left bottom,
        from(#d0d0d0), to(#b0b0b0)
    );
    content: '';
    display: block;
    height: 14px;
    left: 8px;
    position: absolute;
    top: 8px;
    width: 14px;
    -webkit-border-radius: 6px;
    -webkit-box-sizing: border-box;
}


.after:after {
    content: '';
    display: block;
    height: 14px;
    left: 8px;
    position: absolute;
    top: 8px;
    width: 14px;
    -webkit-border-radius: 7px;
    -webkit-box-shadow:
        inset 0 1px 0 rgba(0,0,0,0.25),
        0 1px 0 rgba(255,255,255,0.5);
}

.selected:before {
    background: #10d820 -webkit-gradient(
        radial, 6 6, 4, 6 6, 6,
        from(#10d820), to(#36b300)
    );
    border: 1px inset rgba(0,0,0,0.33);
    height: 14px;
    width: 14px;
}

.selected:after {
    border-width: 2px;
    height: 14px;
    width: 14px;
    -webkit-border-radius: 7px;
}
